# Requesting a new storage allocation on the RDF-Active

Requesting a new storage allocation on the RDF-Active involves the following steps:

## 1. Ensure you have a Research Group in ReCAP

RDF-Active storage is managed through the [ReCAP](../recap/index.md) service. Before you can request a storage allocation, you must have an active Research Group in ReCAP. Please see our guidance on [creating](../recap/creating-a-research-group.md) and [managing](../recap/research-groups.md) a Research Group.

## 2. Confirm you have sufficient RCS Credits

RDF-Active storage allocations must be paid for in advance for a 12-month period, based on the storage quota requested. Before submitting a request, please ensure your Research Group has sufficient [RCS Credits](../recap/rcs-credits.md) to cover the cost of the allocation for the full 12 months.

Details of RDF-Active charging rates are available on the [Charging Structure](https://www.imperial.ac.uk/admin-services/ict/self-service/research-support/rcs/service-offering/charging-structure/) page, and information on purchasing additional credits can be found [here](../recap/rcs-credits.md#buying-rcs-credits). If you require a storage allocation urgently and do not currently have an identified funding source, please [contact us](../support/index.md) to discuss the available options.

## 3. Submit a storage allocation request

Once the above requirements have been met, submit a storage allocation request using the form available [here](https://servicemgt.imperial.ac.uk/esc?id=sc_cat_item&sys_id=611c61043b0d47101feaf49a04e45a02&sysparm_category=52a4a8f21be62110557837b5464bcbd2). We strongly recommend completing the [DART process](https://www.imperial.ac.uk/admin-services/governance/policies-and-guidance/data-assessments/) before submitting your request. If you are unable to do so, please enter "N/A" in the DART reference field on the form.

## 4. Optional: Request migration of existing RDS Projects

If you would like to migrate one or more existing RDS Projects to RDF-Active, you can request this as part of the storage allocation process. When submitting your storage allocation request, attach a file containing a list of the RDS Projects you would like to migrate.

## Need help?

If you are unable to complete any of the steps above, or have any questions about the process, please [contact us](../support/index.md) and we will be happy to assist.
