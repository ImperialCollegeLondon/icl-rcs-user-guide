# Accessing RECAP

!!! warning

    You must be connected to the campus network or otherwise using the [University's remote access service](../remoteaccess.md) in order to connect to the RDF-Active.

To access the portal, open a web browser and go to:

[https://recap.rcs.ic.ac.uk/](https://recap.rcs.ic.ac.uk/)

Select the large **Log In** button. You will be authenticated using Imperial Single Sign-On (SSO).
